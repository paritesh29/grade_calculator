# Student Grade Calculator

This is a simple Python program that calculates a student's grade based on marks (0-100).  
It also shows a positive and encouraging message for each grade.

##  How It Works

1. The program asks for the student's name.
2. It asks for marks (between 0 and 100).
3. It checks if the marks are valid.
4. It calculates the grade using if-elif-else conditions.
5. It displays the grade and a message.

##  Grading System

- 90 - 100  → Grade A
- 80 - 89   → Grade B
- 70 - 79   → Grade C
- 60 - 69   → Grade D
- 0 - 59    → Grade F
